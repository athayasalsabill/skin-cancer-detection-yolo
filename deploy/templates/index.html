Skin Cancer Detection

This is a deep learning model for detecting skin cancer. This model can detect nine types of skin lesions: Actinic Keratosis, Basal Cell Carcinoma, Dermatofibra, Melanoma, Melanocytic Nevus, Pigmented Benign Keratosis, Serborrheic Keratosis, Squamous Cell Carcinoma, Vascular Lesion.
